# Мониторинг клиентов микромаркетов

## Кратко

Каждый день необходимо проводить сбор обратной связи клиентов микромаркетов, заполняя таблицу: https://docs.google.com/spreadsheets/d/1TfKbgTXJKdFUEq2i5eaOP3GFK1O7edsOZri9SDw1C-Q/edit?gid=0#gid=0

В таблице для каждого партнера заведен отдельный лист, в нём указана ссылка на систему, из которой мы будем брать данные для заполнения.

![](image)

Разберем основные данные, вносимые в таблицу для каждой из категорий клиентов:

![](image)


### Первая покупка, клиент 5+ заказов, неудачная покупка и несколько попыток оплаты

Всю информацию о таких клиентах берем из кабинета заказов. Первая покупка и клиент 5+ заказов. Напротив заказа у клиента с первой покупкой будет зеленая метка.

![](image)

У клиента 5+ заказов в карточке будет число заказов, превышающее 5.

![](image)

Неудачная покупка - это клиент, который в итоге так и не купил воду, а несколько попыток оплаты - это несколько заказов у одного клиента, один из которых не получилось оплатить.Напротив заказа

### Отвалившиеся клиенты

Всю информацию о таких клиентах берем в кабинете маркетолога в разделе Отчёты.

![](image)

### Регистрация без заказа

Всю информацию берем из кабинета настроек в разделе Клиенты. Для этого делаем выгрузку и устанавливаем количество заказов, равное 0.

![](image)

При сообщениях используем скрипты, которые закреплены за каждой компанией. В скриптах изменяем данные на корректные (дату и адрес).
