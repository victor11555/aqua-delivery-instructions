# Формирование ссылок для ячеек МикроМаркета

Для того, чтобы сформировать ссылки для ячейки Микромаркета, следует выполнить следующие действия:

1. Перейдите в CRM, в раздел “АкваМаркеты” и выберите нужный шкаф
![](image)

2. Скопируйте ID ячейки.
![](image)

3. Воспользуйтесь ссылкой-шаблоном и сформируйте ссылку на ячейку Микромаркета. Для этого вставьте скомированный ID в ссылку после знака "=" и доменное имя вместо знаков перед ".veel.shop"
Шаблон: https://domain.veel.shop/aquamarket?cell=cellId
Должно получиться следующее: https://domain.veel.shop/aquamarket?cell=1f6c1ad1-7fa9-4797-9a15-0e6b31633a9e
