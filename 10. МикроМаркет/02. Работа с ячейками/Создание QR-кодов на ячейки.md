# Создание QR-кодов на ячейки


## Пошагово
1. Переходим по ссылке для создания кодов – https://qrcode-info.ru/qr-generator/logo
2. Берем ссылки на ячейки из трекшена.
3. Копируем ссылку только так: нажимаем два раза на ячейку, выделяем ссылку и копируем.
![](image)

4. Создаем код: вставляем ссылку в строку и делаем отступ от края 3 стрелки вправо.
![](image)

5. Вставляем изображение для логотипа.
Для этого берем номера для ячеек по ссылке и скачиваем их – https://disk.yandex.ru/d/h8BTIm4C14mjrA
Выбираем нужный номер для кода (например, номер 1 для первой ячейки):

![](image)

6. Сохраняем код и скачиваем в формате png.
7. Перед созданием следующего кода ОБЯЗАТЕЛЬНО обновить страницу.
8. Коды подписываем (например, 21 модуль 1 ячейка) и загружаем на Яндекс. диск (Микромаркеты – папка партнера).
