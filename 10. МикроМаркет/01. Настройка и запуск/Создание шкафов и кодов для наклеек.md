# Создание шкафов и кодов для наклеек


## Пошагово
1. Ставим задачу на Влада Новосельцева, чтобы он создал DeviceId для шкафов. Пример такой задачи: https://pm.appsol.ru/T11302
Дожидаемся, когда в таблице трекшна появятся данные по микромаркетам.

2. Переходим в Кабинет заказов - Микромаркеты и добавляем новый шкаф.
![](image)

3. В открывшемся окне заполняем все данные согласно таблице трекшна, где:
- id устройства - DeviceId
- Метод оплаты - YooMoney СБП
- Число ячеек и номер шкафа указаны в таблице трекшна
- Адрес - Сыктывкар, Первомайская, д. 72 для всех шкафов, после получения микромаркета партнером и установки на точку, адрес необходимо будет заменить.
![](image)

4. и проверяем их (Открываем ячейки с помощью курьерского приложения).
5. Создаем QR-коды для ячеек по инструкции:   Передаем их в отдел маркетинга.
