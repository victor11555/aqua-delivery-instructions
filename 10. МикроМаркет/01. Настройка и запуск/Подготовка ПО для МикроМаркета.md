# Подготовка ПО для МикроМаркета

## Кратко

Для подготовки ПО для МикроМаркетов необходимо знать следующую информацию:

- [ ] Заполненный опросный лист партнера;
- [ ] Количество МикроМаркетов и их расположение: данная информация находится в таблице трекшна запуска МикроМаркетов в разделе "Данные по шкафам".

## Пошагово

1. В кабинете настроек партнера настраиваем офис продаж.
2. Добавляем зону доставки, район и интервалы доставки. Зону можно скопировать из существующего Партнера, район доставки делаем таким же как зону и делаем один интервал равный с 00:00:00 дл 23:59:59 на все дни.
3. Добавляем товары из чек-листа.
З.Ы.: На момент написания инструкции в аквамаркетах не реализована продажа наборов товаров и товары нужно заносить следующим образом:

- Вода 19л. Обязательно указываем тип товара "Вода 19л". Это влияет на то, будет ли клиенту предложено купить или оставить возвратную тару;
- Набор товаров. Каждый товар из этого набора заносим, как отдельный. Самое главное, чтобы эти товары в сумме были равны этому набору;
- Упакованная вода. Заносим этот товар как штучный товар из этой упаковки. Например, товар "Упаковка воды 0.5л 12шт" нужно занести как "Вода 0.5л".
4. Создаем администратора и прописываем его данные в Партнерке для кабинета настроек и кабинета оператора.
![](image)

5. Делаем общие настройки.
Основные первоначальные:

- Название партнера (Должно быть АкваМаркет + Название)
- Часовой пояс
- Прописываем адреса подсайтов и приложений:
URL адрес стола заказов - https://v.**поддомен**.aqua-delivery.ru
URL адрес API - https://api.**поддомен**.aqua-delivery.ru
URL адрес кабинета маркетолога - https://b.**поддомен**.aqua-delivery.ru
URL адрес кабинета настроек - https://admin.**поддомен**.aqua-delivery.ru
URL адрес клиентского сайта 2.0 - https://**поддомен**.veel.shop/

- Заполняем в разделе "Поделится" все поля для ссылок по шаблону - https://**поддомен**.veel.shop/get
![](image)

6. Заполняем раздел электронная почта. Копируем значения из существующего Партнера.
![](image)

7. Переходим в Настройки - Способы оплат и ставим галочку около "Онлайн оплата".
8. Переходим в Настройки - Справочники - Источники заказов и ставим галочку около "aquamarket".
9. В Партнерке переходим в сводные данные и включаем галочки "Маркировка" и "Активен".
10. Добавляем способ оплаты для аквамаркета. Это должен быть из способов онлайн оплаты. Подробнее про добавление способа оплаты описано в инструкции.
11. Подключаем безакцептные платежи;
12. Добавляем кассу и включаем формирование чеков для онлайн оплаты. Данные от кассы берем из старого Партнера. Обязательно отключаем двухэтапную отправку чеков!
![](image)

13. Создаем курьера. Данные фиксируем в чек-листе.
14. Создаем сайт. Как это сделать описано в инструкции.
15. Ставим задачу на Влада, чтобы он создал DeviceId для шкафов.
16. Переходим в кабинет заказов - аквамаркеты и заполняем каждый аквамаркет товарами. Ориентируемся по чек-листу.
17. Создаем форум в телеграмме. Как это сделать описано в инструкции.
18. Формируем ссылки на ячейки и проверяем их (Открываем ячейки с помощью курьерского приложения). После проверки передаем ссылки в отдел маркетинга.
Готово! После всех настроек переходим к тестированию.
