# Настройка сайта 2.0 в Вебмастере

В Вебмастере работаем только под аккаунтом aquab2cnew@yandex.ru.

#### Основные настройки

1. Переходим к нужному сайту и в разделе Индексирование выбираем Рендеринг страниц JavaScript (β).
![](image)

Выбираем Рекомендую рендерить и сохраняем

![](image)

2. Переходим в Настройки. выбираем Привязка к Яндекс Метрике
![](image)

Номер счетчика для сайта находим Яндекс метрике (под аккаунтом aquab2cnew@yandex.ru)

![](image)

Товарный фид

3. Переходим в Кабинет настроек и заполняем url для Сайта 2.0: url должен быть без / в конце
![](image)

4. В товарный фид могут быть добавлены только товары, услуги и акции не должны попасть в товарный фид (Иначе он не пройдет проверку)
Поэтому переходим в раздел с товарами:

![](image)

И переходим к редактированию товара.

![](image)

Нужно выбрать Базовый тип товара: для всего что является товаром - выбираем Товар, для всего что является услугой или акцией - Услуга.

![](image)

Сохраняем

После редактирования товаров, проверить наличие и содержимое товарного фида можно по url:

https://api.domain.aqua-delivery.ru/yandex-feed/catalog.yml, где domain - домен партнера

5. Возвращаемся в Вебмастер, раздел Товары в поиске => Фиды и нажимаем загрузить фид.
![](image)

Заполняем:

- Категория бизнеса - Товары,
- Регионы доставки - Регион, в котором работает доставка,
- Ссылка на файл с предложениями - url Tоварного фида.
![](image)

6. После отправки фида на проверку нужно запустить проверку магазина.
![](image)

7. На этой же странице нужно настроить режим работы доставки.
![](image)

8. Тут нужно заполнить график работы доставки
![](image)
