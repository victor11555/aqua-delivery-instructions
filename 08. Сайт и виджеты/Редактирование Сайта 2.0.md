# Редактирование Сайта 2.0

## Кратко

Перед использованием не забываем выполнить команду aqua update-cli

С помощью этого инструмента можно редактировать meta-теги, подзаголовок и редирект на сайте 2.0.

Команда aqua edit-site имеет следующие флаги:

- --push - автоматическая отправка изменений в gitlab,
- --meta - будут изменены только meta-теги,
- --subtitle - будет изменен/добавлен только подзаголовок,
- --redirect - будет изменен/добавлен только url для редиректа.
Примеры команд:

aqua edit-site --meta --push - отредактирует meta-теги и отправит изменения.

aqua edit-site --subtitle --push - отредактирует подзаголовок сайта и отправит изменения.

aqua delete-site --push - команда для удаления сайта

Использование:

Открываем в VS Code проект client-config, введя команду code documents/client-config в основном терминале

Далее, в терминале VS Code вводим команду aqua edit-site с необходимым набором флагов (Обязательно используем --push).

Заполняем:

- Website Domain - домен сайта (Пример: apps.veel.shop)
- Title meta tag - Meta tag title
- Description meta tag - Meta tag description
- Keywords meta tag - Meta tag keywords
- Subtitle - Subtitle
- Redirect Url - Redirect url
После этого обновленный конфиг будет отправлен в gitlab. Информация на сайте будет обновлена в течении минуты.
