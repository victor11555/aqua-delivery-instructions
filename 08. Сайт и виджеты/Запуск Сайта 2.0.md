# Запуск Сайта 2.0

Перед использованием Aqua-cli не забываем выполнить команду aqua update-cli

Инструкция актуальна для запуска Сайта 2.0 на домене veel.shop

Открываем в VS Code проект client-config: code documents/client-config

Далее, в терминале VS Code вводим команду aqua create-site --push, aqua-cli попросит поочередно вести данные, из которых будет сформирован конфиг. Заполняем:

- Company Type: aqua - для водных компаний, other - для остальных,
- Uid - uid, например из aqua.config.json, (берем из партнерки: настройки партнера-партнер-DeepLink ID)
- Anonymous client token - Токен доступа из Партнерской админки
![](image)

- Select default language (ru / en) - Язык по умолчанию, ru или en,
- Meta tag title - Meta tag title,
- Meta tag description - Meta tag description,
- Meta tag keywords - Meta tag keywords.
Для заполнения Meta Tags можно использовать теги, созданные ранее для Лендинга в Партнерской админке:

![](image)

После заполнения всех данных, конфиг будет отправлен в gitlab и облако. Сайт 2.0 будет доступен по ссылке: uid.veel.shop примерно через 15 минут.
