# Установка тонкого клиента на компьютер партнера

## Кратко

Данная инструкция будет полезна саппорту, который устанавливает тонкий клиент на свое устройство или устройство партнера.
Для начала попросите партнера попробовать установить самостоятельно тонкий клиент, отправив инструкцию https://s.appsol.ru/yVhidbm
Так же предоставьте все данные для заполнения базы данных https://server-1c.service.appsol.ru/(uid партнера)
Пример: https://server-1c.service.appsol.ru/vityazvoda
В первую очередь вам необходимо скачать актуальный тонкий клиент по ссылке: https://disk.yandex.ru/d/9G33b_8edtJckQ
После того как вы скачали файл по ссылке выше, вам необходимо разархивировать (извлечь) в любую вашу папку полученные файлы

![](image)

Далее среди файлов нужно найти файл setup.exe

![](image)

Начнется установка, нажимаем "Далее"

![](image)

Здесь так же "Далее"

![](image)

Язык стоит автоматически, выбираем далее

![](image)

Установка продлится около минуты

![](image)

По итогу вы увидите следующее окно, чтобы не открывать текстовый файл с информацией можно отключить галочку Readme

![](image)

Далее на рабочем столе у вас будет ярлык 1С, его можно открыть и внести необходимые настройки для базы

WARNING: Сейчас всех партнеров включаем через веб-публикацию

![](image)

![](image)

Адрес информационной базы пишется в формате https://server-1c.service.appsol.ru/(uid партнера) Пример на скриншоте выше, так же не забудьте внести имя базы, указав название партнера, чтобы не потеряться в других базах

![](image)

Нажимаем "Готово"

Теперь в базу можно зайти по доступам из битвардена (саппорту на своем компьютере) или по данным партнера, которые создавали для него.

WARNING: Доступами из битвардена на компьютер партнера заходить запрещено
