# Регламент по запуску компаний из проекта "Песочница"

Проект «Песочница» — это программа сотрудничества с компаниями, занимающимися доставкой воды, в рамках которой им предоставляется возможность бесплатно использовать наш сайт и CRM-систему на протяжении 3 месяцев. По окончании этого периода с каждого заказа будет взиматься фиксированная комиссия. Цель проекта — привлечь новых партнёров, демонстрируя удобство и эффективность нашей системы, а также подготовить их к дальнейшему сотрудничеству.

Регламент по настройкам кабинетов для компаний проекта «Песочница»

1. Поэтапное заполнение кабинетов согласно инструкции:
- Все обязательные поля и параметры должны быть корректно настроены и проверены.
- Не проводим какие-то расширенные настройки по типу создания водителей, прописания GTIN, настройки системы лояльности.
2. Для проекта «Песочница» не подключается онлайн-оплата.
3. Все платежи обрабатываются по другим согласованным схемам: наличка, картой курьеру, безналичный по договору.
4. Техническая поддержка для компаний из проекта осуществляется исключительно через электронную почту. Время ответа на запросы — до 1 рабочего дня.
5. Обучение проводится только по функционалу CRM-системы. В чат с партнером из песочницы отправляется ссылка на базу знаний.
6. Предварительный срок запуска кабинета и начала работы по проекту — 14 календарных дней.
7. Партнёрам предоставляются только операторские доступы в кабинет. Администраторские (админские) права не предоставляются в рамках проекта «Песочница».
Этот регламент обязателен для выполнения при работе с каждым партнёром проекта «Песочница» и обеспечивает единый стандарт запуска и сопровождения.
