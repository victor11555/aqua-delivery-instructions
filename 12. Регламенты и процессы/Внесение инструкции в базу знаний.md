# Внесение инструкции в базу знаний

Для внесения инструкции нужно войти на страницу редактирования базы знаний.

1. В расширении битвардена открыть раздел «Хранилище».
![](image)

2. Перейти в раздел «Логин»
![](image)

3. Найти доступ «Редактор Howto» и перейти на страницу
![](image)

4. Затем на сайте войти под доступами editor@aquadelivery.com из битвардена
![](image)

5. Перейти в просмотр базы знаний
![](image)

6. Найти инструкцию и перейти на ее страницу, для редактирования нажать на кнопку «Редактировать запись»
![](image)

7. Редактировать базу знаний нужно в режиме «Визуально», текст редактировать в поле ниже
![](image)

8. Чтобы добавить изображение нужно нажать на кнопку «Добавить медифайл» и загрузить его.
![](image)

9. После редактирования инструкции изменения можно посмотреть по кнопке «Посмотреть изменения», чтобы сохранить изменения нужно нажать на кнопку «Обновить».
![](image)
