# Постановка задач в Яндекс.Трекере

Общий алгоритм создания задачи

1. Нажмите на кнопку Создать задачу в левой части интерфейса.
2. В левом верхнем углу выберите соответствующую очередь (продукт):
Доступные очереди:

- Aqua Delivery - общие системные задачи
- CRM - задачи по кабинету заказов
- 1C - задачи по интеграции с 1С
- Хотелки - пожелания партнеров и команды
- Client App - задачи по клиентскому приложению
- MicroMarket - задачи по микромаркетам
- Veel.Pro - задачи по курьерскому приложению
- DataLens - задачи по аналитике
- Онлайн-оплата - подключение ЮКассы
- ОФД - подключение онлайн-касс.
Если вы ставите багу:

3. Выберите верную очередь в верхнем левом углу.
4. Задайте название задачи по шаблону: Проблема. Продукт. Партнер.
Например: Некорректное значение долга по таре в карточке клиента. CRM. Центр питьевой воды - Темрюк
5. Максимально подробно опишите задачу: приложите скриншоты и иные материалы, которые помогут разобраться в проблеме. Обязательно уточняем у партнёра, когда произошла определённая ситуация (дата и время) - данная информация требуется для упрощения поиска технической информации по возникшим ситуациям.
6. Перейдите к заполнению данных о задаче:
Тип: Задача

Приоритет: Средний стоит по умолчанию, если бага критическая (по ней поступают массовые обращения), то выбираем Критичный.

Автор: указываем свой аккаунт.

Исполнитель: дежурный разработчик.

Наблюдатели: Владимир Колодинов и весь отдел саппорта.

Теги: баги.

Доски: Aqua Delivery.

Спринт: выбираем актуальный.

Дата начала, дедлайн, проект, Story Points - не заполняем.

7. Выбираем Создать задачу.
Если вы ставите хотелку:

8. Выберите очередь Хотелки.
9. Задайте название задачи по шаблону: Пожелание. Партнер.
Например: Добавить в курьерское приложение отчет за период. Кристальная - Волгоград
10. Максимально подробно опишите запрос: приложите скриншоты и иные материалы, которые помогут разобраться в вашем пожелании.
11. Перейдите к заполнению данных о задаче:
Тип: Задача

Автор: указываем свой аккаунт.

Исполнитель: Виктор Забровский.

Наблюдатели: Владимир Колодинов и аккаунт постановщика задачи.

Доски: Хотелки.

Теги, спринт, приоритет, дата начала, дедлайн, проект, Story Points - не заполняем.

12. Выбираем Создать задачу.
Если вы ставите просто задачу (не баг):

13. Выберите верную очередь в соответствии с содержанием задачи.
14. Задайте название задачи по шаблону: Описание задачи. Партнер.
Например: Пробить закрывающие чеки. Танаис - Ростов-на-Дону
15. Максимально подробно опишите запрос: прикрепите скриншоты и иные материалы.
16. Перейдите к заполнению данных о задаче:
Тип: Задача

Автор: указываем свой аккаунт.

Исполнитель: дежурный разработчик.

Наблюдатели: Владимир Колодинов и весь отдел саппорта.

Доски: Aqua Delivery.

Спринт: выбираем актуальный спринт.

Теги, приоритет, дата начала, дедлайн, проект, Story Points - не заполняем.

17. Выбираем Создать задачу.
Если вы ставите задачу по созданию Юкассы:

18. Выберите очередь Онлайн-оплата.
19. Задайте название задачи по шаблону: Способ оплаты Партнер
Например: Юкасса Чимеевская вода или Юкасса СБП Чимеевская вода

20. Пропишите следующие данные:
- [ ] Secret key:
- [ ] ShopID:
- [ ] Чеки на нашей стороне
- [ ] Чеки на стороне партнера
Отметьте галочкой нужный чек-бокс при выборе печати чеков.

21. Перейдите к заполнению данных о задаче:
Тип: Задача

Доски: Онлайн-оплата.

Автор, Исполнитель, Наблюдатели, Спринт, Теги, приоритет, дата начала, дедлайн, проект, Story Points - не заполняем.

22. Выбираем Создать задачу.
Если вы ставите задачу по созданию онлайн-кассы ОФД:

23. Выберите очередь ОФД.
24. Задайте название задачи по шаблону: ОФД Партнер
Например: ОФД Кызырбак - Уфа

25. Пропишите следующие данные:
3 поля из ОФД:
Логин: 
Пароль: 
Код группы:

ИНН: 
Система налогообложения: 
Ставка НДС: 
Агентские признаки: 
Способы оплаты: 
Акцизные товары: 
Резервная почта: 
ФФД: 
Сайт:

26. Перейдите к заполнению данных о задаче:
Тип: Задача

Доски: ОФД.

Автор, Исполнитель, Наблюдатели, Спринт, Теги, приоритет, дата начала, дедлайн, проект, Story Points - не заполняем.

27. Выбираем Создать задачу.
