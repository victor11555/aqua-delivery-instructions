# Пробитие чека коррекции в интерфейсе ОФД

Чек коррекции пробивается для того, чтобы аннулировать чек, пробитый ранее. Самым частым сценарием является пробитие чека коррекции прихода, чтобы аннулировать чек прихода.

1. В первую очередь необходимо перейти в ОФД под данными вашего аккаунта
 https://lk.ofd.ru/
2. Далее перейти во вкладку "Сервис Ferma" и выбрать элемент "Пробить новый чек"
![](image)

3. На вкладке пробития чека в первую очередь выбираем актуальный магазин с кассами.
![](image)

4) Далее необходимо выбрать "Тип чека", который мы создаем. Для того, чтобы выбить чек коррекции на чек прихода нужно выбрать пункт "Чек Коррекции/Возврат прихода"

![](image)

5) Настраиваем налогообложение по чеку (такое же как по чеку, по которому делаете коррекцию, в вашем случае ОСН)

Основание коррекции выбираем "Самостоятельна коррекция"

![](image)

6) Выбираем пункт «Заполнить на основании чека» и ищем чек по его параметрам
 Выбираем ФН

![](image)

Вводим номер фискального документа (его можно взять в чеке, по которому делаем коррекцию) и номер ФПД и нажать «Найти чек», проверить и нажать «Выбрать»

![](image)

![](image)

![](image)

7) Останется только указать только способ оплаты из чека

![](image)

![](image)

8) Пробиваем Чек

![](image)

Готово, пробьется новый чек, который аннулирует предыдущий чек прихода
