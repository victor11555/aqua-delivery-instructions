# Настройка и использование ИС МП

ИС МП - обменно сортовой учет. Инструмент, предназначенный для вывода маркировки в большом количестве. Этот инструмент позволяет выводить маркировку из оборота зная только GTIN воды. После вывода из оборота, на виртуальном складе ЧЗ уменьшается количество маркировок, с указанным GTIN. Этот инструмент подходит для работы со всеми юриками, кроме HoReCa, бюджетников и дистрибьюторов.

Перед настройкой: Договоритесь с Партнером подключиться к его компьютеру на 20 - 30 минут для настройки объемно-сортового учета. Партнер заранее должен подготовить электронную подпись, через которую можно попасть в личный кабинет Честного Знака. Также заранее нужно будет узнать пароль от электронной подписи, если он есть, конечно.

Также заранее запросите у Партнера список маркированной продукции, который продается в нашей системе. Для каждого продукта Партнер должен предоставить GTIN товара.

Подключаемся к Партнеру через AnyDesk. Настройки производим через пользователя администратора, так как нужные доступы есть только у этого пользователя. После окончания работы не забываем выйти из этого пользователя, чтобы Партнер не смог выкачать расширение.

Предупредите Партнера, что для проверки работы функционала, с виртуального склада Партнера будет списана одна маркировка.

# Как работать?

После того, как заказ по договору доставлен, в разделе "Продажи" - "Обмен ИС МП" - "Вывод из оборота ИС МП" попадает запись, которую нужно провести, чтобы маркировка списала с виртуального склада. Стоит отметить, если контрагент является участником обмена по ЭДО с Партнером, то запись о выводе из оборота не будет появляться в разделе ИС МП. С такими клиентами нужно работать через ЭДО!
