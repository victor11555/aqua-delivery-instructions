# Работа с безалкогольными напитками


## Пошагово
1. Создаем заказ с безалкогольными напитками;
![](image)

2. Назначаем водителя на заказ, после чего формируется расходная накладная в непроведенном статусе, после чего вы сможете распечатать документы;
![](image)

3. Водитель доставляет заказ и сканирует всю воду. Обратите внимание, что для безалкогольных напитков не требуется сканирование воды, так как для данной продукции разрешен вывод в объемно-сортовом учете (ОСУ);
4. После доставки заказа, в расходную накладную попадает вся отсканированная маркировка, а также эта расходная накладная проводиться;
5. Далее нам необходимо в ручную заполнить маркировку через ОЗУ. Для этого в расходной накладной нажимаем на "Проверка и подбор" - "Подобрать и проверить безалкогольные напитки" - "Безалкогольные напитки" - "Объемно-сортовой учет" - "Заполнить по документу", после чего вся маркировка будет заполнена по ОСУ. Завершаем подбор.
![](image)

![](image)

# Покупатель работает по ЭДО

Если покупатель работает по ЭДО, то нам будет достаточно отправить документ по ЭДО

![](image)

# Покупатель не работает по ЭДО

В случае, если покупатель по ЭДО не ведет работу, то нам будет необходимо самостоятельно создать вывод из оборота. Обратите внимание, что для воды автоматически создается документ вывода из оборота, однако для безалкогольных напитков нам будет необходимо создать этот документ вручную.

1. Переходим в расходную накладную и нажимаем на кнопку "Вывод из оборота ИС МП";
![](image)

2. В открывшемся окне нажимаем на "Создать вывод из оборота ИС МП" - "Безалкогольные напитки";
![](image)

3. Поля в создаваемом документе должны быть заполнены следующим образом:
  1. Организация - ваша организация, которая произвела реализацию;
  2. Вид продукции - безалкогольные напитки;
  3. Вид учета - объемно-сортовой;
  4. Причина вывода - для собственных нужд предприятия;
  5. Вид первичного документа - прочее;
  6. Наименование документа - Расходная накладная;
  7. Остальные поля должны заполниться автоматически.
![](image)

4. Переходим в раздел "Товары" и выбираем "GTIN" продукции из выпадающего списка (Будет 1 вариант выбора)
![](image)

5. Проводим документ и отправляем его в ЧЗ.
![](image)
