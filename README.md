Остальные разделы без вложенных подпапок или с одним уровнем — см. дерево в репозитории.

## Навигация для ИИ-агентов

1. **Сужение области** — по полю `area` в YAML или по таблице выше выберите верхнюю папку; затем при необходимости подпапку по смыслу запроса (заказы, ОФД, маркировка и т.д.).
2. **Поиск по формулировке** — используйте `title`, заголовки `#` / `##` и `tags`; синонимы пользователя могут не совпадать с текстом статей — проверяйте несколько близких файлов в том же разделе.
3. **Ответ** — при опоре на инструкцию указывайте **относительный путь к файлу** в репозитории; если в контенте есть скриншоты без текстового описания, явно отмечайте неполноту шагов.
4. **Конфликты и дубли** — при двух противоречивых статьях отдавайте приоритет более узкой тематической папке и более свежему контексту (при сомнении — сообщите о противоречии).
